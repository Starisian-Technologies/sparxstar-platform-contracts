# Code of Conduct

## Our Standard

Starisian Technologies expects contributors and reviewers in this repository to communicate professionally, stay factual, and focus on repository quality, safety, and governance.

## Expected Behavior

- be respectful and precise
- keep feedback technical and actionable
- disclose conflicts or uncertainty directly
- protect private security information
- respect the repository's licensing and governance boundaries

## Unacceptable Behavior

- harassment, discrimination, or abusive conduct
- sharing confidential or security-sensitive information publicly
- misrepresenting ownership, approval, or repository authority
- pressuring maintainers to bypass governance or licensing rules

## Enforcement

Starisian Technologies repository owners may remove content, restrict participation, or close interactions that violate this standard.

## Reporting

If conduct in this repository needs review, contact the repository owners through established project channels or open a private maintainer contact if available.
