# Support

## Where to Ask for Help

Use GitHub issues in this repository for:

- questions about contract intent or documentation
- requests for clarification about repository structure or governance
- issues with repository automation or metadata files

## Before Opening an Issue

- confirm whether the change belongs in a source service repository rather than here
- include the contract path involved
- describe the consumer impact clearly

## What This Repository Does Not Support Directly

- implementation-specific bugs in downstream consumers
- runtime support for services that merely import these contracts
- urgent security disclosures

For security matters, use the process in [SECURITY.md](SECURITY.md).
